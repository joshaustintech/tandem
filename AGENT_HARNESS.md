# Agent harness

Use this repo for short, repeatable implementation loops.

## Loop

1. Read `PLAN.md`.
2. Read current source files before editing.
3. Pick one smallest task.
4. State the exact scope in one sentence.
5. Edit only required files.
6. Run the smallest proof command.
7. Update this harness if the task changes repo state.
8. Stop after one task.
